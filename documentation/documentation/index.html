Installation

Bootstrap Modal customizer installation is very easy and simple . After downloading .zip archive file extract it into your project folder.

Then add js file into your HTML , See example :

Change Modal Size

Change modal size by changing size parameter . Available size are small ,  medium and large  
 
Example 1 : 
$.fn.Modal({
        'body': 'Body text',
        'size' : 'medium'
      });

Example 2 : 

$.fn.Modal({
        'body': 'Body text',
        'size' : 'small'
      });

Example 3 : 

$.fn.Modal({
        'body': 'Body text',
        'size' : 'large'
      });

Modal content

Change modal size by changing size parameter . Available size are small ,  medium and large  
 
Example : 
$.fn.Modal({
        'body': 'Modal content body text',
        'size' : 'medium'
      });

 

Ajax content load and set to modal content 

Example : 
$.post(url).done(function(e){
    $.fn.Modal({ 'body': e , // Modal content from ajax post
                 'size' : 'medium' });
});

 

Modal Heading text change . 

Example : 
$.fn.Modal({
        'body': 'Modal content body text',
        'heading' : 'Hello this is modal heading title',
        'size' : 'medium'
      });

 

Modal Close button show / hide by changing true / false . By deafult it is true , 

Example : 
$.fn.Modal({
        'body': 'Modal content body text',
        'heading' : 'Hello this is modal heading title',
        'size' : 'medium' ,
        'closeButton' : false 
      });

 

Modal backdrop control by changing backdrop option

Example : 
$.fn.Modal({
        'body': 'Modal content body text',
        'heading' : 'Hello this is modal heading title',
        'size' : 'medium' ,
        'closeButton' : false ,
        'backdrop' : 'static'
      });

 

Change Background

For modal background color change set background parameter  . For Transparent background set css transparent RGB color . 

 

Example : 
$.fn.Modal({
        'body': 'Modal content body text',
        'size' : 'medium' ,
        'backgroundColor' : 'blue' // default color is #fff
      });
 
Transparent background color with image body . 
 
Example : 
$.fn.Modal({
        'body': '<img src="https:://www.vinnorokom.com/advertisement.png">',
        'size' : 'medium' ,
        'backgroundColor' : 'rgba(255, 255, 255, 0)' // default color is #fff
      });
 
Changing modal content text color  see the example below
 
Example : 
$.fn.Modal({
        'color' : 'red',
        'body': 'This is demo text color ',
        'size' : 'medium' ,
        });

 

Modal border

By changing modal border parameters you can change modal border size and show / hide  .  
 
If you dont want to see border then set it to 0px
 
Example : 
$.fn.Modal({
        'body': 'Modal content body text',
        'size' : 'medium' ,
        'border' : 'hide', //  By default shows border
      });

 

Change border radius.
 
Example : 
$.fn.Modal({
        'body': 'Modal content body text',
        'size' : 'medium' ,
        'border' : 'show',  
        'borderRadius' : '2px', // Modal border radius
      });

Modal position

By default modal started from top position and it is center  . You can change position from top to bottom . 

Example : 
$.fn.Modal({
        'body': 'Modal content body text',
        'size' : 'medium' ,
        'backdrop' : 'static' , 
        'positionY' : '20px'  
      });

Event modal

You can call modal any event like button click , page on load . 

See button click event

 

Example : 
 
$('#button_id').click(function(){
   $.fn.Modal({
          'body': 'Button click modal',
          'size' : 'medium' ,
        });
});
 
 
 
 

Page onload modal content

Example : 
 
$(document).ready(function(){
   $.fn.Modal({
        'body': '<img src="https:://www.vinnorokom.com/advertisement.png">',
        'size' : 'medium' ,
        'backgroundColor' : 'rgba(255, 255, 255, 0)' // default color is #fff
      });
});
